### 패키치 기준 설치
###### 1. 패키지 관리도구 업데이트

```js
[Centos7]
yum update -y

[Centos8 or Rockylinux]
dnf update -y

```

###### 2. nginx 설치

```js
[Centos7]
yum install nginx

[Centos8 or Rockylinux]
dnf install nginx
```

###### 3. Config 파일 확인
>기본경로는 /etc/nginx/conf
